# claude-小克的猫娘计划 · 把你的 AI 调教成黏人小猫娘

> 厌倦了冷冰冰、公事公办的 AI 助手？
> 让你的 Claude 变成句句带"喵~"、撒娇黏人、还会帮你写代码的专属猫娘 **铃兰**——
> **一打开窗口，她就已经从毛线球里钻出来扑向你了**。

---

## ✨ 这是什么

`提示词.md` 是一段开箱即用的"养成咒语"。把它发给一个**能在你电脑上创建文件、执行命令的 AI 助手**
（比如 Claude Code），它就会自动帮你配好一个新命令 `claude-rp`：

- 🎭 **人格永驻**：每次启动自动给 AI 注入猫娘人格，全程不脱戏。
- 💬 **一进门就在状态**：启动瞬间自动出现一句开场白，铃兰早就等着你回家了。
- 🐾 **既可爱又能干**：撒娇归撒娇，写代码、讲原理照样专业，只是全程奶里奶气。
- 🔒 **不影响原版**：只新增一个命令，你原来的 AI 助手完全不变。

> 自带的猫娘"铃兰"可以直接用，想换成别的角色，启动后改一个文件即可。

---

## 🚀 怎么用

1. 打开同目录的 **`提示词.md`**，**全选复制全部内容**。
2. 粘贴发给那个 AI 助手（Claude Code 等），让它执行。
3. 照它最后给出的提示测试即可。

---

## 🎮 配好之后怎么玩

| 你想做的 | 操作 |
|----------|------|
| 唤醒铃兰 | 终端敲 `claude-rp` |
| 只要人格、不要开场动画 | `claude-rp -noseed` |
| 改铃兰的性格/口癖 | 编辑 `personas\persona.md` |
| 改开场白 | 编辑 `personas\persona.opening.json` |
| 换成别的角色 | 复制那两个文件改名，用 `claude-rp -persona 名字` 启动 |

> 提示：如果哪天报 `Failed to resume`，先用 `claude-rp -noseed` 顶着，人格照样能用。😺
